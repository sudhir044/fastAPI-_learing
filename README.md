student magagenten API with FastAPI 
 A simple REST API built using FastAPI and Pydantic for learing basic of backend so we can use  of AI model in our development 

 Fatures 
 GET all students
 POST a new student 

 Tech Stack 
 python 3
 FastAPI 
 Uvicorn
 Pydantic

Installation
Clone the repository:

git clone https://github.com/sudhir044/fastapi-student-api.git
Navigate to the project folder:
cd fastapi-student-api

Install dependencies:
pip install -r requirements.txt
Run the server:

uvicorn main:app --reload

The API will start at:

http://127.0.0.1:8000
